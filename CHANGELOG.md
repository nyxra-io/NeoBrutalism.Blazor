# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.1.1] — 2026-04-21

### Added

- `HideArrows` parameter on `NBCarousel` to hide navigation arrows.

## [1.1.0] — 2026-04-13

### Added

- `NBHero` component.
- `NBSection` component.
- `NBTimeline` component.
- `NBFooter` component.
- `NBFeatureCard` component.
- `NBStatCard` component.

## [1.0.0] — 2026-04-12

### Added

- Initial public open-source release.
- 30+ Neo Brutalism components for Blazor.
- CSS design token system with `NBThemeProvider`.
- Tailwind CSS build pipeline.
- Blazor WASM documentation site with live demos.
- DocFX API reference generation.
